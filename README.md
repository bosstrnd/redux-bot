# TRND Redux — бот доступа

Бот выдаёт доступ к **TRND Redux**. Работает в одном из двух режимов, которые
переключают админы.

## Режимы
1. **Проверка подписки** (по умолчанию) — пользователь подписывается на каналы
   [@ceogtamedia](https://t.me/ceogtamedia) и [@igoresh1x](https://t.me/igoresh1x),
   жмёт «Проверить» → бот проверяет подписку и выдаёт доступ.
2. **Личный промокод** — пользователь присылает скриншот ввода промокода,
   админ одобряет (✅) или отклоняет (❌). При одобрении выдаётся доступ.

## Админы
Переключают режим командой **/mode**:
- @ceoprmanager
- @igoreshqa_w

Они же видят заявки на скриншоты и жмут «Одобрить»/«Отклонить».

## Установка
1. Создай бота у [@BotFather](https://t.me/BotFather), вставь токен в `.env`.
2. **Добавь бота администратором в оба канала** `@ceogtamedia` и `@igoresh1x` —
   без этого бот не сможет проверять подписку.
3. Заполни `.env`:
   - `BOT_TOKEN` — токен бота.
   - `REDUX_ACCESS_TEXT` — сообщение со ссылкой/инструкцией доступа к Редуксу.
   - `REVIEW_CHAT_ID` — *необязательно*: ID приватной группы для проверки заявок.
     Если оставить пустым, заявки приходят в личку админам (каждый админ должен
     один раз нажать /start, чтобы бот узнал его ID).
4. Установи и запусти:
   ```powershell
   pip install -r requirements.txt
   python bot.py
   ```
   Или дважды кликни `start.bat`.

## Важно
- Для режима подписки бот **обязан быть админом** обоих каналов.
- Чтобы заявки (скриншоты) доходили до админов в личку, админы должны
  предварительно запустить бота (/start). Надёжнее — создать приватную группу,
  добавить туда бота и указать её ID в `REVIEW_CHAT_ID`.
- Файлы `.env`, `state.json`, `admins.json` в гит не попадают (см. `.gitignore`).
- На хостинге BotHost данные храните в `/app/data` — бот сам берёт путь из
  переменной `DATA_DIR`, так режим и список админов переживут перезапуск.

## Совместные розыгрыши (добавление каналов)
Список обязательных каналов можно менять на лету:
- `/channels` — показать текущие каналы-условия.
- `/addchannel @канал` — предложить **совместный розыгрыш**: бот (он должен быть
  админом в этом канале) пишет в личку админам канала-партнёра запрос с кнопкой
  «✅ Согласиться». После согласия канал добавляется в обязательные подписки.
  Партнёр должен предварительно запустить бота (/start), иначе запрос не дойдёт.
- `/removechannel @канал` — убрать канал из условий.

## Команды
- `/start` — начать (выдаёт инструкцию по текущему режиму).
- `/mode` — (только админы) переключить режим.
- `/channels`, `/addchannel`, `/removechannel` — (только админы) управление каналами.
